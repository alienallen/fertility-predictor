# 备孕预测应用 - UI 设计规范

## 色彩系统

| 用途 | 颜色名 | 色值 |
|-----|--------|------|
| 主色 | 鼠尾草绿 | `#9CAF88` |
| 辅助色 | 赤陶色 | `#D4A574` |
| 强调色 | 玫瑰粉 | `#E8B4B8` |
| 背景 | 奶油白 | `#FDF8F3` |
| 文字 | 深暖灰 | `#4A4541` |
| 次要文字 | 浅暖灰 | `#9B9B9B` |
| 安全期 | 浅绿 | `#C5D4B8` |

## 概率曲线配色

| 概率区间 | 颜色 |
|---------|------|
| 高 (>60%) | `#9CAF88` 鼠尾草绿 |
| 中等 (30-60%) | `#D4A574` 赤陶色 |
| 低 (<30%) | `#E8B4B8` 玫瑰粉 |

## 字体

- **标题/数字**: `"PingFang SC", "Helvetica Neue", sans-serif`
- **正文**: `"PingFang SC", "Microsoft YaHei", sans-serif`
- **概率数字**: 超大加粗（56-72px）

## 圆角

- 小元素（标签、按钮）: `8px`
- 卡片: `16px`
- 大卡片/弹窗: `24px`

## 组件规范

### 按钮

| 类型 | 样式 |
|-----|------|
| 主按钮 | 背景 `#9CAF88`，白色文字，圆角 8px |
| 次按钮 | 白色背景，`#9CAF88` 边框，`#9CAF88` 文字 |
| 危险按钮 | 背景 `#E8B4B8`，白色文字 |

### 卡片

- 背景：白色 `#FFFFFF`
- 圆角：16px
- 内边距：24px

### Tab 切换

- 默认：文字 `#4A4541`
- 选中：背景 `#9CAF88`，文字白色

### 输入框

- 背景：`#FDF8F3`
- 边框：2px solid `#E5E0DC`
- 聚焦：边框变 `#9CAF88`
